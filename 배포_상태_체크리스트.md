## 🔧 즉시 조치 필요 사항

### 우선순위 1: 환경 변수 설정

Vercel 대시보드에서 다음 환경 변수들을 설정해야 합니다:

```bash
# Vercel CLI로 설정하는 방법
cd muyi-giftcard
vercel env add CHLIFES_GENID production
vercel env add CHLIFES_GIFTNM production
vercel env add CHLIFES_ENC_KEY production
vercel env add CHLIFES_IV production
vercel env add NEXT_PUBLIC_CHLIFES_URL production
vercel env add WIZZPAY_MID production
vercel env add WIZZPAY_IV_KEY production
vercel env add WIZZPAY_SALT production
vercel env add WIZZPAY_PASSWORD production
vercel env add NEXT_PUBLIC_WIZZPAY_URL production
vercel env add NEXT_PUBLIC_SUPABASE_URL production
vercel env add NEXT_PUBLIC_SUPABASE_ANON_KEY production
vercel env add SUPABASE_SERVICE_ROLE_KEY production
```

또는 Vercel 대시보드에서:
1. https://vercel.com 접속
2. 프로젝트 `muyi-giftcard` 선택
3. Settings → Environment Variables
4. 위의 환경 변수들을 모두 추가

### 우선순위 2: 변경사항 커밋 및 푸시

현재 많은 변경사항이 커밋되지 않았습니다:

```bash
# 변경사항 확인
git status

# 변경사항 추가
git add .

# 커밋
git commit -m "규격서 준수: API 엔드포인트 및 요청 형식 수정"

# 푸시
git push origin main
```

### 우선순위 3: 재배포

환경 변수 설정 후 재배포:

```bash
# 자동 배포 (GitHub 푸시 시)
git push origin main

# 또는 수동 배포
vercel --prod
```
