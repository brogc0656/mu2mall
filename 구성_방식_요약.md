# 구성 방식 요약

## 현재 구성: PM2 + 웹 기반 모니터링 대시보드 ✅

### 실행 중인 서비스
1. **프록시 서버** (포트 3001)
   - PM2로 실행 중
   - 클라이프스 API 프록시

2. **모니터링 대시보드** (포트 3002)
   - PM2로 실행 중
   - 웹 브라우저에서 접속: http://localhost:3002
   - 시각적 모니터링 및 제어 가능

### Docker 구성
- ✅ 파일 준비됨 (`Dockerfile`, `docker-compose.yml`)
- ⚠️ 아직 실행하지 않음
- 필요 시 즉시 실행 가능
